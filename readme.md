# TherianusMulticines 🎬🍿

Proyecto de la asignatura de **Diseño de Interfaces** para 2º de DAM/DAW. Este proyecto consiste en el desarrollo de la interfaz web para un cine, construida en su totalidad utilizando **Tailwind CSS**.

## 🚀 Características del Proyecto

La página web simula la experiencia de usuario de un cine real e incluye las siguientes funcionalidades:

* **Página Principal (Index):** Muestra las películas más taquilleras y las novedades destacadas.
* **Horarios:** Consulta de las sesiones y proyecciones disponibles para cada día.
* **Detalles de la Película:** Ficha técnica, sinopsis y panel de información de cada cinta.
* **Comentarios:** Sección donde los usuarios pueden leer las reseñas de las películas.

## 🛠️ Tecnologías Utilizadas

* HTML5
* Tailwind CSS
* Node.js / npm (para la gestión de paquetes y el servidor de desarrollo).

## ⚙️ Instalación y Ejecución

Sigue estos pasos para descargar y ejecutar el proyecto en tu máquina local:

1. **Descargar el proyecto:** Clona este repositorio o descarga el archivo ZIP y extráelo en tu ordenador.

2. **Instalar las dependencias:**
   Abre una terminal en la carpeta raíz del proyecto y ejecuta el siguiente comando para instalar Tailwind y el resto de dependencias necesarias:
   ```bash
   npm install
3. **Ejecutar el entorno de desarrollo:**
   Una vez instaladas las dependencias, levanta el servidor local ejecutando:
    ```bash
    npm run dev
